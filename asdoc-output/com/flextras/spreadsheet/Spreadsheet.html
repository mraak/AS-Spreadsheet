Packagecom.flextras.spreadsheet
Classpublic class Spreadsheet
InheritanceSpreadsheet PaintGrid mx.controls.DataGrid
Implements ISpreadsheet

Spreadsheet allows you to visauly and programatically use features and calculations found in Excel and other Spreadsheet programs. It is divided into rows and columns and can accept other objects and collections as variables in the calculations.

Default MXML PropertydataProvider
