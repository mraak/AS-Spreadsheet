Classpublic class SpreadsheetEvent
InheritanceSpreadsheetEvent flash.events.Event

SpreasheetEvent carries various information based on the type. Most important, there are ERROR, and WARNING types. - ERROR: In this case, something has happened that will stop the correct execution of the requested procedure. - WARNING: Something has happened that will result in some loss of data but the procedure will run normally.
