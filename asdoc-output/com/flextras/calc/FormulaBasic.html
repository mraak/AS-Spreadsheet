Classpublic class FormulaBasic
InheritanceFormulaBasic Object

This class contains static functions for solving wide range of formulas, as found in other popular spreadsheet applications. This class contains set of basic formulas, other formulas can be found in other classes like FormulaConst and FormulaLogic. This class does not need to be implemented, it relies on static methods. Supported forumlas are: SUM, AVERAGE, MAX, MIN, ROUND, FLOOR, CEIL, COUNT, MEDIAN

public static function solve(formula:String, args:Array):String

Main function to solve any formula.

Parameters

formula:String — Formula that you wish to solve. You can use static constants such as FormulaBasic.SUM
 
args:Array — Array with arguments to the specific formula

Returns
String — String representing the numeric solution to the formula

Example
The following code calculates the sum of numbers 1, 2, and 3. 
         
         var result:String = FormulaBasic.solve(FormulaBasic.SUM, [1,2,3]);
         trace(result); // 6
