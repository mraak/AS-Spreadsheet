Packagecom.flextras.calc
Classpublic class ControlObject
InheritanceControlObject flash.events.EventDispatcher

ControlObject is an essential class when working with Calc. You don't need to access it directly but Calc uses it internally extensivelly. Each ControlObject 'controls' one element of the calculation logic, like TextField or Spreadsheet cell. This element that is under control is referenced by ControlObject.ctrl property.
