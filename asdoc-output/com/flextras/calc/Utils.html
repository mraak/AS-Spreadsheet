Utils class contains helper methods for various operations regarding Spreadsheet, mathematical and logical operations. User of Spreadsheet SDK does not need to directly use these functions, however some advanced users might find them very useful in wide variety of applications.

isString()method 
public static function isString(input:String):Boolean

Recognizes if supplied input is supposed to be treated as string within calculation expressions. That means it checks wheather it is enclosed in double quotes. This is not to be confused with ActionScript String class, it is a separate construct that is used in Calc. Example: isString("s"), returns false isString(""s""), returns true

Parameters

input:String — String that needs to be checked if it is a valid string representation for Calc.

Returns
Boolean — Boolean true if the specified string is a valid string representation for Calc.

public static function repairOperators(exp:String):String

This function attempts to repair the misstyped and redundant operators, by returning only the first valid operator. e.g.: -+5++/3+, result: -5+3. Secondly, it attempts to repair redundant operators in parenthesis, e.g.: (+6/), result: (5+6).

Returns

resolveFieldsRange()method 
public static function resolveFieldsRange(fields:String):Array

Resolves range specified as string into array of field id's. It works on cell id's such as a1, a2, b1,.. as well as on collections registered by Calc.

Parameters

fields:String — String representing the range of fields/cells to resolve.

Returns
Array — Array containing the ID's of fields/cells in the range.

Example
The following resolves range on one column specified by collon 
     
     var range:Array = Utils.resolveFieldsRange("a1:a4");
     trace(range); // ["a1", "a2", "a3", "a4"]
The following resolves range on one row specified by collon 
     
     var range:Array = Utils.resolveFieldsRange("a1:c1");
     trace(range); // ["a1", "b1", "c1"]
The following resolves range on one row specified by comma 
     
     var range:Array = Utils.resolveFieldsRange("a1,b1,c1");
     trace(range); // ["a1", "b1", "c1"]
The following resolves square range on multiple rows and columns specified by collon 
     
     var range:Array = Utils.resolveFieldsRange("a1:c2");
     trace(range); // ["a1", "a2", "b1", "b2", "c1", "c2"]
